# GhostDraw Installer Resources

## Required Image Files

### banner.bmp
- **Size**: 493 × 58 pixels
- **Format**: 24-bit BMP
- **Content**: GhostDraw logo + "Setup Wizard" text
- **Background**: Dark cyberpunk theme with pink/cyan gradients

### dialog.bmp
- **Size**: 493 × 312 pixels
- **Format**: 24-bit BMP
- **Content**: Cyberpunk-themed background with ghost/pencil imagery
- **Style**: Match the settings window aesthetic

## How to Create

1. Use any image editor (Paint.NET, GIMP, Photoshop)
2. Create images with exact dimensions above
3. Export as 24-bit BMP format
4. Place in this `Resources/` directory

## Temporary Workaround

For testing, you can use solid color BMPs:
- Banner: Dark background with app name
- Dialog: Dark background (matching #0A0E27)

The installer will build without these, but they enhance the user experience.
